Проект содержит тесты на открытие страниц Отус.

Настроен автоматический запуск job в Jenkins:
- по push новых изменений в git-репозиторий
- по расписанию (в час ночи)

После прогонки тестов:
- происходит архивация артифактов
- генерируются Allure отчеты
- происходит отправка уведомления в Slack-канал
- происходит отправка письма на почту gmail
- выполняется backup job и её настроек с помощью SCM Sync configuration plugin

Уведомления содержат в себе:
- статус сборки
- название job в Jenkins
- номер билда
- название ветки git
- время выполнения job
- количество тестов (всего / упавших / пропущенных / пройденных)
- ссылку на job

В корне проекта приложен файл "Jenkins_demo.wmv", показывающий процесс запуска и работы job.
